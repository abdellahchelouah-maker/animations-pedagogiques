Applications pédagogiques en Visual Basic

Cette page regroupe des programmes pédagogiques développés en Visual Basic ainsi qu’une série de QCM web associés. Ils permettent de travailler le vocabulaire, l’orthographe, les automatismes et des notions d’électrotechnique à l’aide du générateur de QCM Pro.

Les applications Visual Basic se lancent sous Windows, en utilisant le générateur de QCM Pro et les packs ZIP ci‑dessous, tandis que les QCM web se déroulent directement dans le navigateur.

Installation et utilisation

Commencez par installer les deux kits Visual Basic via les programmes d’installation ci‑dessous. Ensuite, décompressez tous les fichiers ZIP (applications, QCM et aides) dans le dossier de l’application QCM Pro, puis lancez l’installation en double‑cliquant sur setup.exe présent dans ce dossier.

Les packs ZIP contiennent l’ensemble des fichiers nécessaires (application, données QCM et fichiers d’aide) et doivent être extraits avant utilisation.

Pré‑requis : kits d’installation Visual Basic

Kits Visual Basic pour Windows

Installez ces deux kits Visual Basic avant d’utiliser QCM Pro et les applications associées. Ils installent les composants nécessaires au bon fonctionnement des programmes sous Windows 10/11.

Applications principales (vb1 à vb3)

vb1 Découvre les mots

Application destinée à travailler le vocabulaire et l’orthographe, en distinguant les majuscules des minuscules, à partir de listes de mots adaptées au niveau des élèves.

Télécharger le pack (.zip)

vb2 Corrige les dictées

Des dictées truffées de fautes à corriger. L’application aide à repérer rapidement les erreurs récurrentes, à suivre la progression des élèves et à travailler la relecture attentive.

Télécharger le pack (.zip)

vb3 Générateur de QCM – QCM Pro

Application permettant de générer des QCM autocorrectifs à partir d’une base de fichiers alimentée par les QCM techniques ci‑dessous (packs ZIP dédiés).
